## Gitの基本的コマンド
* リポジトリを初期化
```bash
$ git init
```

* リポジトリ状態を確認
```bash
$ git status
```

* ステージ領域へファイルを追加
```bash
$ git add
```

* リポジトリの歴史を記録
```bash
$ git commit
```

* コミットログを確認
```bash
$ git log
```

* 変更差分を確認
```bash
$ git diff
```
